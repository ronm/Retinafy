Retinafy
=========

Adds max-width = naturalWidth/w to images based on selector. Also watches for additional images with a mutation observer.


Usage
------

```js
import Retinafy from './retinafy'
```

or

```html
<script type="text/javascript" src="retinafy.js"></script>
```

then

```js
new Retinafy(".retina");
```
